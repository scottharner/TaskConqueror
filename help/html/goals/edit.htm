Editing A Goal

The Edit Goal window can be used to create a new goal or edit an existing goal. In order to create a goal, simply enter a value in the Title textbox and click the Save button. To edit a goal, modify a value in one of the window's controls, and click the Save button.

To access the Edit Goal window, click on the Goals tab in the Main Menu window, select a goal from the list, and click the New or Edit button.

Edit Goal Window Controls

Save

Clicking this button saves the changes made in the Edit Goal window.

Close

Clicking this button closes the Edit Goal window.

Add

Clicking this button provides access to the Add Projects window, for the purpose of adding existing projects to the goal.

New

Clicking this button provides access to the Edit Project window, for the purpose of creating a new project.

Edit

Clicking this button opens the selected project in the Edit Project window, for the purpose of editing the project.

Delete

Clicking this button will permanently delete the selected project from the Task Conqueror application.

View

Clicking this button provides access to the Help window.

Title

Entering text in this field will result in the value being used as the goal's title. This field is mandatory.

Status

Selecting a value in this field will result in the value being used as the goal's status. This field is mandatory.

Category

Entering text in this field will result in the value being used as the goal's estimated cost. This value conveys how much money it will take to complete a goal. This field is optional.

Date Created

This label displays the date and time when the goal was created.

Date Completed

This label displays the date and time when the goal was completed, if ever.

Projects

This list displays the projects that are contained by the goal, if any.
