Adding Projects To A Goal

The Add Projects screen can be used to add existing projects to a goal. In order to add a project, simply select a project from the unassigned projects list, click the Add button, and click the OK button.

To access the Add Projects screen, click on the Goals tab in the Main Menu window, and then click on the Edit button on the Edit Goal window.

Add Projects Window Controls

Help

Clicking this button provides access to the Help window.

Unassigned Projects

This list contains all of the projects in the application that have not been completed, abandoned, or assigned to a goal. The projects are listed alphabetically.

Add

Clicking this button will add the selected project from the Unassigned Projects list to the Selected Projects list.

Remove

Clicking this button will remove the selected project from the Selected Projects list, returning it to the Unassigned Projects list.

Selected Projects

This list contains all of the projects that have been selected for addition to the goal.

OK

Clicking this button will add any projects in the Selected Projects list to the goal. Additionally the current window will be closed.

Cancel

Clicking this button will close the window without adding any projects to the goal.
