Browsing And Searching Goals

The Goals tab can be used to browse existing goals and search for specific goals. In order to search, simply enter a value in the Title Contains textbox and click the Filter button. To restore the goal listing to its original complete state, clear the Title Contains textbox and click the Filter button again.

To access the Goals tab, click on the Goals tab in the Main Menu window.

Goals Tab Controls

New

Clicking this button provides access to the Edit Goal window, for the purpose of creating a new goal.

Edit

Clicking this button opens the selected goal in the Edit Goal window, for the purpose of editing the goal.

Delete

Clicking this button will permanently delete the selected goal from the Task Conqueror application.

View

Clicking this button provides access to the Help window.

Title Contains

Entering text in this field will cause the goal listing to be filtered based upon title when clicking the Filter button. The filtered results will include any goals that contain the entered characters in their title. Partial matches are included in the results.

Filter

Clicking this button will filter the list of goals presented on the Goals tab. Results of the filtering will be determined by the values provided in the Title Contains textbox.

First

Clicking this button displays the first twenty goals.

Previous

Clicking this button displays the previous twenty goals.

Next

Clicking this button displays the next twenty goals.

Last

Clicking this button displays the last twenty goals.

Sort By

Selecting a value from this combobox will cause the goal listing to be sorted based upon the selected value when clicking the Sort button.

Sort

Clicking this button sorts the goal list by the field selected in the Sort By combobox.
