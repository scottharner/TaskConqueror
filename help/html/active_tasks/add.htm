Adding An Active Task

The Add Active Task screen can be used to add an existing task to the active task list, marking the task as active. In order to add a task, simply select a task from the inactive tasks tree, click the Add button, and click the OK button.

To access the Add Active Task screen, click on the Main Menu window, and then click on the Add button on the Active Tasks tab.

Add Active Tasks Window Controls

View

Clicking this button provides access to the Help window.

Inactive Tasks

This tree contains all of the inactive tasks in the application that have not been completed or abandoned. The tasks are organized by their containing projects and goals. Tasks not contained by a project appear under the Unassigned node.

Add

Clicking this button will add the selected task from the Inactive Tasks tree to the Selected Tasks list.

Remove

Clicking this button will remove the selected task from the Selected Tasks list, returning it to the Inactive Tasks tree.

Selected Tasks

This list contains all of the tasks that have been selected for addition to the active tasks list.

OK

Clicking this button will add any tasks in the Selected Tasks list to the active task list, marking the tasks as active. Additionally the current window will be closed.

Cancel

Clicking this button will close the window without adding any tasks to the active task list.
