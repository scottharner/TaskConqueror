Browsing And Searching Active Tasks

The Active Tasks tab can be used to browse existing active tasks and search for specific active tasks. In order to search, simply enter a value in the Title Contains textbox and click the Filter button. To restore the active task listing to its original complete state, clear the Title Contains textbox and click the Filter button again.

To access the Active Tasks tab, click on the Active Tasks tab in the Main Menu window.

Active Tasks Tab Controls

Add

Clicking this button provides access to the Add Active Task window.

Edit

Clicking this button opens the selected task in the Edit Task window.

Remove

Clicking this button will remove the selected task from the active tasks list. The task will be set to inactive.

View

Clicking this button provides access to the Help window.

Title Contains

Entering text in this field will cause the active task listing to be filtered based upon title when clicking the Filter button. The filtered results will include any active tasks that contain the entered characters in their title. Partial matches are included in the results.

Filter

Clicking this button will filter the list of active tasks presented on the Active Tasks tab. Results of the filtering will be determined by the values provided in the Title Contains textbox.

First

Clicking this button displays the first twenty active tasks.

Previous

Clicking this button displays the previous twenty active tasks.

Next

Clicking this button displays the next twenty active tasks.

Last

Clicking this button displays the last twenty active tasks.

Sort By

Selecting a value from this combobox will cause the active task listing to be sorted based upon the selected value when clicking the Sort button.

Sort

Clicking this button sorts the active task list by the field selected in the Sort By combobox.
