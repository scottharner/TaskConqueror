Main Menu

The main menu window provides a starting point from which all the major modules of Task Conqueror may be accessed. Clicking on any one of these tabs will redirect the user to the associated module.

Active Tasks

This option can be used to search, add, remove, and update active tasks.

Tasks

This option can be used to search, add, update, and delete tasks.

Projects

This option can be used to search, add, update, and delete projects that can be composed of one or more tasks.

Goals

This option can be used to search, add, update, and delete goals that can be comprised of one or more projects.

Reports

This option can be used to generate various reports regarding tasks, projects, and goals.

Administrative

This option provides facilities for cleaning up completed and abandoned tasks, projects, and goals.
