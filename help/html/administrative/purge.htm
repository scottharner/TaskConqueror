Purging Tasks, Projects, And Goals

The Administrative tab can be used to purge existing tasks, projects, and goals from the Task Conqueror application. In order to purge, simply select the object type(s) from the Object Type combobox, check the statuses from the Status checkboxes, and click the Purge button. Purged objects cannot be restored.

To access the Administrative tab, click on the Administrative tab in the Main Menu window.

Administrative Tab Controls

Purge

Clicking this button removes objects of the selected object type(s) and statuses from the Task Conqueror application.

Help

Clicking this button provides access to the Help window.

Object Type

Selecting an option from this combobox will cause objects of the selected type(s) to be purged when clicking the Purge button.

Status

Checking these options will cause objects of the checked statuses to be purged when clicking the Purge button.
