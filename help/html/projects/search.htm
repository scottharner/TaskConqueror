Browsing And Searching Projects

The Projects tab can be used to browse existing projects and search for specific projects. In order to search, simply enter a value in the Title Contains textbox and click the Filter button. To restore the project listing to its original complete state, clear the Title Contains textbox and click the Filter button again.

To access the Projects tab, click on the Projects tab in the Main Menu window.

Projects Tab Controls

New

Clicking this button provides access to the Edit Project window, for the purpose of creating a new project.

Edit

Clicking this button opens the selected project in the Edit Project window, for the purpose of editing the project.

Delete

Clicking this button will permanently delete the selected project from the Task Conqueror application.

View

Clicking this button provides access to the Help window.

Title Contains

Entering text in this field will cause the project listing to be filtered based upon title when clicking the Filter button. The filtered results will include any projects that contain the entered characters in their title. Partial matches are included in the results.

Filter

Clicking this button will filter the list of projects presented on the Projects tab. Results of the filtering will be determined by the values provided in the Title Contains textbox.

First

Clicking this button displays the first twenty projects.

Previous

Clicking this button displays the previous twenty projects.

Next

Clicking this button displays the next twenty projects.

Last

Clicking this button displays the last twenty projects.

Sort By

Selecting a value from this combobox will cause the project listing to be sorted based upon the selected value when clicking the Sort button.

Sort

Clicking this button sorts the project list by the field selected in the Sort By combobox.
