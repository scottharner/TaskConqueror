Editing A Project

The Edit Project window can be used to create a new project or edit an existing project. In order to create a project, simply enter a value in the Title textbox and click the Save button. To edit a project, modify a value in one of the window's controls, and click the Save button.

To access the Edit Project window, click on the Projects tab in the Main Menu window, select a project from the list, and click the New or Edit button.

Edit Project Window Controls

Save

Clicking this button saves the changes made in the Edit Project window.

Close

Clicking this button closes the Edit Project window.

Add

Clicking this button provides access to the Add Tasks window, for the purpose of adding existing tasks to the project.

New

Clicking this button provides access to the Edit Task window, for the purpose of creating a new task.

Edit

Clicking this button opens the selected task in the Edit Task window, for the purpose of editing the task.

Delete

Clicking this button will permanently delete the selected task from the Task Conqueror application.

Title

Entering text in this field will result in the value being used as the project's title. This field is mandatory.

Goal

This label displays the name of the goal that the project is associated with, if any.

Status

Selecting a value in this field will result in the value being used as the project's status. This field is mandatory.

Estimated Cost

Entering text in this field will result in the value being used as the project's estimated cost. This value conveys how much money it will take to complete a project. This field is optional.

Date Created

This label displays the date and time when the project was created.

Date Completed

This label displays the date and time when the project was completed, if ever.

Tasks

This list displays the tasks that are contained by the project, if any.
