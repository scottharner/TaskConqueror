Adding Tasks To A Project

The Add Tasks screen can be used to add existing tasks to a project. In order to add a task, simply select a task from the unassigned tasks list, click the Add button, and click the OK button.

To access the Add Tasks screen, click on the Projects tab in the Main Menu window, and then click on the Edit button on the Edit Project window.

Add Tasks Window Controls

View

Clicking this button provides access to the Help window.

Unassigned Tasks

This list contains all of the tasks in the application that have not been completed, abandoned, or assigned to a project. The tasks are listed alphabetically.

Add

Clicking this button will add the selected task from the Unassigned Tasks list to the Selected Tasks list.

Remove

Clicking this button will remove the selected task from the Selected Tasks list, returning it to the Unassigned Tasks list.

Selected Tasks

This list contains all of the tasks that have been selected for addition to the project.

OK

Clicking this button will add any tasks in the Selected Tasks list to the project. Additionally the current window will be closed.

Cancel

Clicking this button will close the window without adding any tasks to the project.
