Overview

Task Conqueror is a task management application with a focus on goal-centric task completion. It provides facilities for tracking progress on goals, projects, and tasks, and helps the user easily create an active tasks list based on outstanding goals.

If you should encounter any problems while using the application, please contact taskconqueror@gmail.com.
