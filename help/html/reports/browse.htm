Browsing Reports

The Reports tab can be used to browse existing reports.

To access the Reports tab, click on the Reports tab in the Main Menu window.

Reports Tab Controls

Run

Clicking this button provides access to either the Run Report window or the Print Report window, for the purpose of printing a paper copy of the selected report.

Help

Clicking this button provides access to the Help window.

Active Tasks

Selecting this item and clicking the Run button opens the Print Report window, for the purpose of printing a paper copy of the report.

Completed Goals

Selecting this item and clicking the Run button opens the Run Report window, for the purpose of running the report.

Completed Projects

Selecting this item and clicking the Run button opens the Run Report window, for the purpose of running the report.
