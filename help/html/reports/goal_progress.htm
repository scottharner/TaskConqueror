Running The Goal Progress Report

The Run Report window can be used to provide criteria for a report and launch the Print Report window. In order to provide criteria for the Goal Progress Report, simply select a goal from the Goal combobox and click the Run button.

To access the Run Report window, click on the Reports tab in the Main Menu window, select the Goal Progress Report from the list, and click the Run button.

Run Report Window Controls

Run

Clicking this button opens the Print Report window.

Help

Clicking this button provides access to the Help window.

Goal

Selecting a goal in this field will result in the value being used as the goal for which the report will be run. The report will include projects within the selected goal.
