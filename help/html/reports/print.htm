Printing A Report

The Print Report window can be used to print a paper copy of a report. In order to print a report, simply click the Print button from the Print Report window.

To access the Print Report window, click on the Reports tab in the Main Menu window, select a report from the list, and click the Run button. Depending on the report, you may need to provide criteria on the Run Report window and click the Run button.

Print Report Window Controls

Print

Clicking this button prints a paper copy of the report.

Help

Clicking this button provides access to the Help window.
