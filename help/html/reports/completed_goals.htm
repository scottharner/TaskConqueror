Running The Completed Goals Report

The Run Report window can be used to provide criteria for a report and launch the Print Report window. In order to provide criteria for the Completed Goals Report, simply enter a starting date in the Start Date date picker, enter an ending date in the End Date date picker, and click the Run button.

To access the Run Report window, click on the Reports tab in the Main Menu window, select the Completed Goals Report from the list, and click the Run button.

Run Report Window Controls

Run

Clicking this button opens the Print Report window.

View

Clicking this button provides access to the Help window.

Start Date

Selecting a date in this field will result in the value being used as the starting date when running the report. The report will include goals with a completion date on or after this date, but no later than the End Date.

End Date

Selecting a date in this field will result in the value being used as the ending date when running the report. The report will include goals with a completion date before or on this date, but no sooner than the Start Date.
