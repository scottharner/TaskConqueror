Editing A Task

The Edit Task window can be used to create a new task or edit an existing task. In order to create a task, simply enter a value in the Title textbox and click the Save button. To edit a task, modify a value in one of the window's controls, and click the Save button.

To access the Edit Task window, click on the Active Tasks tab in the Main Menu window, select a task from the list, and click the Edit button. Alternatively, click on the Tasks tab in the Main Menu window and click the New button.

Edit Task Window Controls

Save

Clicking this button saves the changes made in the Edit Task window.

Close

Clicking this button closes the Edit Task window.

Title

Entering text in this field will result in the value being used as the task's title. This field is mandatory.

Project

This label displays the name of the project that the task is associated with, if any.

Status

Selecting a value in this field will result in the value being used as the task's status. This field is mandatory.

Priority

Selecting a value in this field will result in the value being used as the task's priority. This field is mandatory.

Active

Checking or unchecking this box will result in the value being used to indicate whether or not the task is being actively worked on. This field is mandatory.

Date Created

This label displays the date and time when the task was created.

Date Completed

This label displays the date and time when the task was completed, if ever.
